CARA MENGECHECK IP DENGAN COMMAND LINE

Cara cek IP komputer lewat CMD yang pertama adalah melalui fitur Command Prompt (CMD).
Melalui fitur ini, kamu akan melihat IP dengan semacam perintah khusus. Untuk lebih jelasnya, simak langkah-langkahnya di bawah ini :

  1. Klik Start, lalu ketik Command Prompt di fitur pencarian.
  2. Setelah halaman Command Prompt terbuka, ketik ipconfig dan tekan Enter.
  3. Alamat IP akan terlihat pada baris ‘IPv4 Address’.
